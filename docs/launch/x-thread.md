## Tweet 3 (How It Works)

```
How the swarm thinks:

1. Fetch live SPX + VIX data
2. Each agent analyzes through their lens
3. Round 1: Independent opinions
4. Round 2-3: See others' views, change minds (or dig in)
5. Extract consensus: direction, confidence, agreement %
6. Detect herding + contrarian signals
7. Synthesize a specific trade card

The magic: when 20/24 agents agree at 80%+ conviction — pay attention.
```

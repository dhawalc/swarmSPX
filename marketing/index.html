How it works

A 24-agent swarm runs every signal through 3 rounds of debate. Each agent has a specialty — VWAP, dealer gamma, options flow, FOMC, sentiment, internals. The debate produces a trade card. The risk gate decides if it dispatches.

[brain]

24 Specialist Agents

Technical (VWAP, gamma, momentum) · Macro (FOMC, VIX, GEX) · Sentiment (flow, fear, contrarian) · Strategists (calendar, spread, sizing). Each is a distinct LLM persona that votes on every cycle.

[chart]

Live Dealer Gamma

Built-in GEX engine computes per-strike dealer gamma, gamma flip, call wall, put wall — from free CBOE OI + Schwab Greeks. Replaces a $199/mo SpotGamma subscription.

[shield]

Pre-Trade Risk Gate

Every trade passes 8 checks: kill switch, daily/weekly/monthly loss bands, consecutive losses, position cap, data freshness, idempotency. ~50ms budget. Fail-loud, never silent.

[target]

Kelly Position Sizer

Fractional Kelly with daily lock. Position size written to disk at session open, immutable mid-session. No discretionary overrides. Discipline that lives in code, not willpower.

[paper]

Paper Broker

Every signal opens a shadow position. Tracks target / stop / EOD exits via real option chain lookup. 30 days of paper data answers "does this make money?" without risking a dollar.

[radar]

Real-time Dashboard

Vanilla canvas neural-network viz. Watch the 24 agents debate live. ELO leaderboard ranks them by historical accuracy per regime. Telegram alerts on high-confidence signals.
